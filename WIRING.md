# Схема подключения ESP32 Piano Trainer

## Компоненты

- ESP32 DevKit v1 (30 pin)
- TFT дисплей ILI9341 2.4" 240x320 с тачскрином XPT2046
- WS2811 LED лента (54 светодиода)
- **MAX98357A I2S DAC (аудиовыход)**
- Резистор 220Ω (для DATA линии LED)
- Конденсатор 1000μF (для питания LED)
- **Динамик 4Ω 3W (или наушники 32Ω)**

## Подключение TFT дисплея (ILI9341 + XPT2046)

```
TFT ILI9341          ESP32
-----------          -----
VCC                  3.3V    (пин 3V3)
GND                  GND     (пин GND)
CS                   GPIO 5  (пин 18)
RST                  GPIO 17 (пин 33)
DC/RS                GPIO 16 (пин 32)
MOSI                 GPIO 23 (пин 23)
SCK                  GPIO 18 (пин 19)
MISO                 GPIO 19 (пин 22)

XPT2046 (Touch)
-----------
VCC                  3.3V
GND                  GND
CS                   GPIO 19 (пин 22)
IRQ                  GPIO 25 (пин 25)
MOSI                 GPIO 23 (общий с TFT)
MISO                 GPIO 19 (общий с TFT)
SCK                  GPIO 18 (общий с TFT)
```

## Подключение WS2811 LED ленты

```
WS2811 LED           ESP32
-----------          -----
+5V                  5V 2A  (внешнее питание!)
GND                  GND   (общая земля с ESP32)
DATA                 GPIO 2 (пин 4)
                     |
                   220Ω
                     |
                   DATA IN
```

**Важно:** Питание LED ленты должно быть отдельным! ESP32 не может обеспечить достаточный ток.

## Подключение MAX98357A I2S DAC

```
MAX98357A          ESP32
-----------        -----
VCC                3.3V    (пин 3V3)
GND                GND     (пин GND)
BCLK               GPIO 26 (пин 12)
LRCL/LRC           GPIO 27 (пин 13)
DIN                GPIO 14 (пин 11)
SD                 GND     (режим I2S)
FILT               не подключать
Gain               GND     (через резистор, опционально)
```

**Примечания:**
- Пин SD (Shutdown) должен быть подключен к GND для работы в режиме I2S
- Пин FILT можно не подключать
- Для регулировки громкости используйте встроенный потенциометр или измените volume в коде

## Подключение динамика

```
MAX98357A          Динамик
-----------        -------
OUT+               + динамика
OUT-               - динамика
```

**Важно:** 
- Используйте динамик 4Ω - 8Ω, 3W или меньше
- Для наушников используйте 32Ω наушники с разъемом 3.5мм

## Полная схема питания

```
Блок питания 5V 2A
      │
      ├─────── +5V ─────── WS2811 VCC
      │
      ├─────── GND ─────── WS2811 GND
      │                     │
      │                     └────── ESP32 GND (общая земля!)
      │
      └─────── 5V ──[AMS1117 3.3V]── 3.3V ── TFT VCC
```

## Детальное подключение

### TFT разъём (14 pin)

```
Пин  TFT          ESP32
1    VCC    ────── 3.3V
2    GND    ────── GND
3    CS     ────── GPIO 5
4    RST    ────── GPIO 17
5    DC/RS  ────── GPIO 16
6    MOSI   ────── GPIO 23
7    SCK    ────── GPIO 18
8    MISO   ────── GPIO 19
9    LED_A  ────── 3.3V (через 10Ω для яркости)
10   SDI    ────── не подключать
11   SDO    ────── не подключать
12   T_CS   ────── GPIO 19
13   T_DIN  ────── GPIO 23
14   T_DO   ────── GPIO 19
15   T_IRQ  ────── GPIO 25
16   T_CLK  ────── GPIO 18
```

### WS2811 подключение

```
ESP32 GPIO 2 ──[220Ω]──┬── DATA IN (WS2811)
                       │
GND ───────────────────┴── GND (WS2811)

+5V 2A ─────────────────── +5V (WS2811)
```

### MAX98357A подключение

```
ESP32                MAX98357A
-----                ---------
3.3V   ────────────── VCC
GND    ────────────── GND
GPIO 26 ───────────── BCLK
GPIO 27 ───────────── LRC
GPIO 14 ───────────── DIN
GND    ────────────── SD (Shutdown)

MAX98357A            Динамик
---------            -------
OUT+   ────────────── + динамика
OUT-   ────────────── - динамика
```

**Примечание:** 
- Используйте резистор 220-470Ω на линии DATA для защиты LED
- Добавьте конденсатор 1000μF между +5V и GND рядом с LED лентой
- Общая земля обязательна между ESP32 и питанием LED

## Маппинг светодиодов на ноты

```
LED 0   = A0    (MIDI 21)  - Синий
LED 1   = A#0   (MIDI 22)  - Красный
LED 2   = B0    (MIDI 23)  - Синий

LED 3   = C1    (MIDI 24)  - Зелёный
LED 4   = C#1   (MIDI 25)  - Красный
LED 5   = D1    (MIDI 26)  - Зелёный
LED 6   = D#1   (MIDI 27)  - Красный
LED 7   = E1    (MIDI 28)  - Зелёный
LED 8   = F1    (MIDI 29)  - Зелёный
LED 9   = F#1   (MIDI 30)  - Красный
LED 10  = G1    (MIDI 31)  - Зелёный
LED 11  = G#1   (MIDI 32)  - Красный
LED 12  = A1    (MIDI 33)  - Зелёный
LED 13  = A#1   (MIDI 34)  - Красный
LED 14  = B1    (MIDI 35)  - Зелёный

LED 15  = C2    (MIDI 36)  - Жёлтый
... (продолжается по той же схеме)

LED 51  = B4    (MIDI 71)  - Белый
LED 52  = C5    (MIDI 72)  - Белый
LED 53  = C#5   (MIDI 73)  - Красный
```

## Цветовая схема

| Октава | Ноты      | Белые клавиши | Чёрные клавиши |
|--------|-----------|---------------|----------------|
| 0      | A0-B0     | Синий         | Красный        |
| 1      | C1-B1     | Зелёный       | Красный        |
| 2      | C2-B2     | Жёлтый        | Красный        |
| 3      | C3-B3     | Оранжевый     | Красный        |
| 4      | C4-C5     | Белый         | Красный        |

## Предупреждения

⚠️ **Не подключайте LED ленту к 3.3V ESP32!** Только к внешнему 5V.

⚠️ **Общая земля обязательна** между ESP32 и внешним питанием.

⚠️ **Используйте резистор** на линии DATA для защиты первого светодиода.

⚠️ **Добавьте конденсатор** для сглаживания скачков напряжения.
