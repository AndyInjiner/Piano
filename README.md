# 🎹 ESP32 Piano Trainer

Обучающее устройство для игры на пианино на базе ESP32 с TFT дисплеем и LED лентой WS2811.

## 📋 Описание

Устройство визуализирует ноты из MIDI файлов на светодиодной ленте (54 светодиода = 54 клавиши пианино) и позволяет обучаться игре через визуальную подсветку клавиш.

## 🔧 Компоненты

| Компонент | Модель | Примечание |
|-----------|--------|------------|
| Микроконтроллер | ESP32 DevKit v1 | WiFi + Bluetooth |
| Дисплей | ILI9341 240x320 | TFT с сенсорным экраном XPT2046 |
| LED лента | WS2811 | 54 светодиода |
| Питание | 5V 2A | Для ESP32 и LED ленты |

## 🔌 Схема подключения

Подробная схема подключения находится в файле [WIRING.md](WIRING.md).

### Краткая распиновка

```
ESP32          Компонент
-----          ---------
GPIO 2         WS2811 DIN
GPIO 5         TFT CS
GPIO 16        TFT DC
GPIO 17        TFT RST
GPIO 18        TFT SCLK
GPIO 19        TFT MISO / TOUCH CS
GPIO 23        TFT MOSI
GPIO 25        TOUCH IRQ
3.3V           TFT VCC, TOUCH VCC
5V             WS2811 VCC
GND            Общий земля
```

### WS2811 LED Mapping

```
Светодиод 0  =  A0 (MIDI нота 21)
Светодиод 1  =  A#0 (MIDI нота 22)
...
Светодиод 53 =  C5 (MIDI нота 74)
```

## 🛠️ Установка и настройка

### 1. Подготовка Arduino IDE

1. Установите Arduino IDE (1.8.19+)
2. Добавьте ESP32 board manager:
   - File → Preferences → Additional Boards Manager URLs
   - Добавьте: `https://raw.githubusercontent.com/espressif/arduino-esp32/gh-pages/package_esp32_index.json`
3. Установите платы ESP32:
   - Tools → Board → Boards Manager
   - Найдите "ESP32" и установите

### 2. Установка библиотек

Установите следующие библиотеки через Library Manager:

- `TFT_eSPI` (Bodmer)
- `FastLED`
- `WebSockets` (Links2004)
- `ArduinoJson` (version 6)

**Важно:** Скопируйте файл `User_Setup.h` в папку библиотеки TFT_eSPI:
```
Documents/Arduino/libraries/TFT_eSPI/User_Setup.h
```

### 3. Загрузка прошивки

1. Откройте `piano.ino` в Arduino IDE
2. Выберите плату: Tools → Board → ESP32 → "ESP32 Dev Module"
3. Выберите порт: Tools → Port
4. Нажмите Upload

### 4. Android приложение

1. Откройте Android Studio
2. Создайте новый проект
3. Скопируйте файлы из папки `android/`
4. Соберите APK: Build → Build Bundle(s) / APK(s) → Build APK(s)
5. Установите на Android устройство

## 🎮 Использование

### Шаг 1: Конвертация MP3 в MIDI

1. Посетите https://basicpitch.spotify.com
2. Загрузите MP3 файл
3. Скачайте полученный MIDI файл

### Шаг 2: Подключение к ESP32

1. Включите ESP32
2. На Android устройстве подключитесь к WiFi сети "PianoTrainer"
3. Пароль: `piano2024`

### Шаг 3: Загрузка MIDI файла

**Через Android приложение:**
1. Откройте приложение Piano Trainer
2. Нажмите "📁 Загрузить MIDI файл"
3. Выберите MIDI файл на устройстве

**Через веб-интерфейс:**
1. Откройте браузер
2. Перейдите на http://192.168.4.1
3. Загрузите MIDI файл через форму

### Шаг 4: Воспроизведение

1. Выберите файл из списка
2. Нажмите PLAY
3. LED лента будет подсвечивать ноты
4. Используйте slider для изменения скорости

### Кнопки на TFT дисплее

- **PLAY** - Начать воспроизведение
- **STOP** - Остановить воспроизведение
- **SEL** - Выбрать файл (карусель)

## 🎹 Цветовая схема LED

| Октава | Цвет белых клавиш | Цвет чёрных клавиш |
|--------|-------------------|-------------------|
| 0 (A0-B0) | Синий | Красный |
| 1 (C1-B1) | Зелёный | Красный |
| 2 (C2-B2) | Жёлтый | Красный |
| 3 (C3-B3) | Оранжевый | Красный |
| 4 (C4-C5) | Белый | Красный |

## ⚙️ Технические характеристики

- **Диапазон нот:** A0 (21) до C5 (74) - 54 ноты
- **Скорость воспроизведения:** 0.1x - 2.0x
- **Поддерживаемые форматы:** Standard MIDI File (.mid, .midi)
- **Максимум файлов:** Зависит от размера LittleFS (~3MB)

## 🔧 Troubleshooting

### TFT дисплей не работает
- Проверьте правильность подключения пинов
- Убедитесь, что файл `User_Setup.h` скопирован правильно

### LED лента не светится
- Проверьте питание (5V, достаточный ток)
- Проверьте подключение GPIO 2

### Ошибки при загрузке MIDI
- Убедитесь, что файл в формате MIDI (не MP3!)
- Проверьте размер файла (должен помещаться в LittleFS)
